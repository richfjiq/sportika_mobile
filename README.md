## Setup

```
git clone git@github.com:richfjiq/sportika_mobile.git
```

```
add .env file

API_URL='https://sportika-server.herokuapp.com/api'
GOOGLE_CLIENT_ID=
```

```
node version 14.19.0
```

```
yarn
```

```
cd ios
pod install
cd ..
```

```
yarn start
```

ios emulator

```
npx react-native run-ios
```

android emulator

```
npx react-native run-android
```
